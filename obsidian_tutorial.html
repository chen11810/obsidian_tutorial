Obsidian 完整使用教程

从零开始到高效知识管理 · 初级 + 进阶两阶段学习路径

1. 安装与初始设置
下载安装 · 创建 Vault · 基础界面熟悉
  • 1
    下载 Obsidian
    前往 obsidian.md 下载对应平台版本（Windows / macOS / Linux / iOS / Android），Obsidian 个人使用完全免费。
  • 2
    创建 Vault（知识库）
    Vault 本质上是本地文件夹。建议将 Vault 放在 iCloud / Dropbox / OneDrive 中以实现多设备同步。启动后点击「Create new vault」选择文件夹即可。
  • 3
    认识界面三大区域
    左侧栏文件列表、搜索、书签
    中间区编辑器主区域，支持多标签/分屏
    右侧栏反向链接、大纲、标签面板
  • 4
    推荐初始设置
    Settings → Editor：开启「Readable line length」保证阅读舒适度；开启「Vim key bindings」（可选）；主题推荐 Minimal 或 Things。
2. Markdown 核心语法
标题 · 加粗 · 列表 · 代码块 · 表格
# 一级标题 ## 二级标题 ### 三级标题 **加粗文字** *斜体文字* ~~删除线~~ `行内代码` - 无序列表 - 第二项 - 嵌套子项 1. 有序列表 2. 第二项 > 引用块文字 | 列1 | 列2 | 列3 | |-----|-----|-----| | 数据 | 数据 | 数据 | ```python print("代码块") ```
Obsidian 同时支持「实时预览」和「源码」两种编辑模式，用 Ctrl+E 切换。Live Preview 模式下 Markdown 会即时渲染，推荐日常使用。
3. 双向链接（核心功能）
内部链接 · 创建新笔记 · 关系图谱
  • 1
    创建内部链接
    输入 [[笔记名]] 即可链接到另一篇笔记。输入 [[ 后会自动弹出已有笔记列表，支持模糊搜索。
  • 2
    链接别名
    使用 [[实际文件名|显示文字]] 让链接显示不同的文字，如 [[Python编程|Python]]。
  • 3
    链接到章节
    使用 [[笔记名#标题]] 精确链接到某篇笔记的特定章节。
  • 4
    查看关系图谱
    点击左侧「Graph View」图标，可视化查看所有笔记的链接关系。孤立节点意味着笔记还未建立连接，需要整理。
双向链接的核心思维：先写内容，后整理结构。不需要预先设计完美的文件夹结构，链接会自然形成知识网络。
4. 文件管理与标签
文件夹组织 · Tag 标签 · 搜索过滤
  • 1
    文件夹策略（PARA法）
    Projects当前进行中的项目
    Areas持续关注的领域
    Resources参考资料库
    Archive归档的历史内容
  • 2
    使用标签
    在笔记中输入 #标签名 即可添加标签。支持嵌套标签如 #读书/小说，标签面板可统一管理所有标签。
  • 3
    强大的搜索功能
    Ctrl+Shift+F 全局搜索，支持 tag:#标签名、path:文件夹、file:文件名 等高级过滤语法。
5. 嵌入与附件
图片 · PDF · 音频 · 嵌入其他笔记
![[图片文件名.png]] ← 嵌入图片 ![[笔记名]] ← 嵌入其他笔记内容 ![[笔记名#章节标题]] ← 嵌入特定章节 ![[PDF文件.pdf|宽度]] ← 嵌入PDF，可指定宽度 直接拖拽图片到编辑区即可插入
Settings → Files & Links → Default location for new attachments，建议设置为「In subfolder under current folder」统一管理附件。
1. Properties / YAML 元数据
结构化信息 · 自定义字段 · Dataview 基础
在笔记最顶部添加 YAML Front Matter：
--- title: 我的读书笔记 author: 作者名 date: 2024-01-15 status: 读中 tags: [读书, 小说, 科幻] rating: 4 --- 正文内容从这里开始...
Obsidian 1.4+ 版本引入了可视化的 Properties 面板（Ctrl+;），无需手动编辑 YAML，直接点击添加字段即可。结合 Dataview 插件可进行强大的数据库式查询。
2. 模板系统
日记模板 · 项目模板 · Templater 插件
  • 1
    开启内置模板功能
    Settings → Core plugins → Templates 开启，设置模板文件夹路径（如 Templates/）。使用 Ctrl+Shift+T 插入模板。
  • 2
    日记模板示例
    # {{date:YYYY-MM-DD}} {{time:dddd}} ## 今日目标 - [ ] ## 今日记录 ## 反思与总结 **亮点：** **待改进：** ## 明日计划 - [ ]
  • 3
    Templater 插件（进阶）
    支持 JavaScript 动态模板，如自动插入当前日期、随机引用、从选中文字创建笔记等高级操作。
3. Dataview 插件
把 Vault 当数据库 · 动态列表 · 表格查询
常用查询示例：
```dataview LIST FROM #读书 WHERE status = "读中" SORT date DESC ``` ```dataview TABLE author, rating, date FROM "Books" WHERE rating >= 4 SORT rating DESC ``` ```dataview TASK FROM "Projects" WHERE !completed GROUP BY file.name ```
Dataview 查询是实时的，笔记内容改变时自动刷新。非常适合管理读书清单、项目追踪、习惯记录等需要聚合视图的场景。
4. 必装插件推荐
社区精选 · 功能扩展 · 使用场景
Dataview
把笔记变成数据库，支持 DQL 查询语言
Templater
高级模板系统，支持 JS 脚本动态生成

日历视图管理日记，一键跳转到特定日期
Kanban
看板视图管理任务，支持拖拽排序

手绘风格白板，直接嵌入笔记中

周记/月记/年度回顾，配合 Calendar 使用

5. 卡片盒笔记法（Zettelkasten）
原子化笔记 · 永久笔记 · 链接思想
  • 1
    闪念笔记（Fleeting Notes）
    随时随地快速记录想法，不加修饰，放入 Inbox 文件夹，定期整理（每天/每周）。
  • 2
    文献笔记（Literature Notes）
    阅读书籍/文章时，用自己的话总结要点，记录来源信息。一本书对应一篇笔记。
  • 3
    永久笔记（Permanent Notes）
    将闪念和文献笔记提炼为独立的、原子化的思想单元。每篇笔记只讲一个概念，并主动与其他永久笔记建立链接。
  • 4
    MOC（内容地图）
    Map of Content：为某个主题创建索引笔记，汇集该主题下所有相关笔记的链接，作为知识领域的导航中心。
每日工作流
晨间回顾 · 日记记录 · 晚间整理
  • 🌅
    早晨（5分钟）
    打开 Calendar 插件创建今日日记，查看昨日未完成任务，用 Dataview 查看所有「进行中」项目，设定今日 3 个核心目标。
  • ☀️
    日间（随时）
    遇到好的想法 → 快速记入 Inbox；阅读文章 → 复制核心段落到对应笔记；会议记录 → 创建新笔记，链接到相关项目。
  • 🌙
    晚间（15分钟）
    清空 Inbox：将今天的闪念分类整理；完成今日日记的反思部分；为明天预设任务；图谱检查是否有孤立笔记需要链接。
读书笔记工作流
读前 · 读中 · 读后 · 复习
书名：原则 作者：瑞·达利欧 状态：读中 评分： 开始时间：2024-01-10 完成时间： 标签：[商业, 管理, 自传]
  • 1
    读前：创建书籍笔记
    使用书籍模板，填写基本信息。写下读这本书的目的和预期收获。
  • 2
    读中：高亮与摘录
    使用 Kindle Highlights 插件自动导入标注；或手动记录关键段落，务必用自己的话改写，而非直接复制。
  • 3
    读后：提炼与链接
    写核心观点总结（3-5条）；与已有笔记建立链接；创建行动清单；更新状态为「已读」。
项目管理工作流
项目笔记 · 任务追踪 · 进度看板
--- project: 网站重设计 status: 进行中 deadline: 2024-03-01 team: [小明, 小红] --- ## 项目目标 ## 里程碑 - [x] 需求调研 - [ ] 原型设计 (截止 2/15) - [ ] 开发实现 - [ ] 测试上线 ## 相关资料 - [[竞品分析报告]] - [[用户调研结果]] - [[设计规范]] ## 会议记录 - [[2024-01-15 项目启动会]]
配合 Kanban 插件，可以把 [[项目名]] 卡片在「待办/进行中/完成」列之间拖拽，非常直观。
多设备同步方案
官方同步 · iCloud · Git 备份
  • 💰
    Obsidian Sync（官方，$8/月）
    最无缝，支持版本历史，端对端加密。适合对隐私有要求、不想折腾的用户。
  • 🆓
    iCloud（仅 Apple 设备免费）
    Mac + iPhone 组合最推荐。将 Vault 放在 iCloud Drive 文件夹下，自动同步，偶尔有冲突需手动解决。
  • 🔧
    Obsidian Git（免费，需动手）
    Vault 存为私有 Git 仓库，配合 Obsidian Git 插件自动提交推送。跨平台，有版本历史，但需要一点 Git 基础。

命令面板（Ctrl+P）
最强导航 · 无需记忆快捷键
命令面板是 Obsidian 最强大的功能之一，几乎所有操作都可以在这里完成：
  • 模糊搜索所有命令
    输入关键词即可找到对应操作，如输入「graph」找到图谱相关命令，输入「split」找到分屏命令。
  • 固定常用命令
    鼠标悬停在命令上可以看到「固定」选项，固定后下次打开命令面板会显示在最前面。
